# Система управления библиотекой (Library Management System)
Микросервисное приложение для автоматизации работы библиотеки: управления каталогом книг, процессом бронирования и выдачи

## Архитектура системы
Проект построен на микросервисной архитектуре с разделением ответственности:

- Auth Service (Порт: 8080): Управление пользователями, регистрация, аутентификация (JWT) и роли (RBAC).
- Core Service (Порт: 8081): Бизнес-логика библиотеки (книги, выдачи, бронирования).
- Frontend (React + Vite): Клиентское приложение для читателей и персонала.

## Ролевая модель

В системе реализовано три уровня доступа:

1. READER (Читатель): * Просмотр каталога книг.
   - Бронирование доступных книг (/my эндпоинты).
   - Просмотр своей истории займов и активных броней.

2. LIBRARIAN (Библиотекарь):
   - Управление каталогом книг (добавление, редактирование, удаление).
   - Обработка заявок на бронирование (подтверждение/отклонение).
   - Оформление выдачи книг и фиксация возвратов.

3. ADMIN (Администратор):
   - Все права библиотекаря.
   - Управление пользователями через панель управления (создание аккаунтов библиотекарей).

## Технологический стек

1. Backend
   - Java 21/Kotlin
   - Spring Boot 3.x
   - Spring Security + JWT
   - Spring Data JPA (PostgreSQL)
   - Liquibase (миграции базы данных)
   - MapStruct (маппинг DTO и сущностей)

2. Frontend
   - React 18
   - Vite (сборка проекта)
   - Axios (HTTP-клиент)
   - React Router Dom (навигация и защита маршрутов)
